= 檢視儲存庫
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
「庫位」清單會顯示所選子租戶的現有庫位。

. 若要檢視清單、請從功能表中選取*物件儲存>桶*。
. 如有需要、請使用子租戶下拉式功能表選取其他子租戶。
+
顯示每個儲存桶的簡單資訊、例如：

+
** 名稱
** 區域
** 尺寸
** 建立日期
** 物件
** 行動（例如刪除庫位）
