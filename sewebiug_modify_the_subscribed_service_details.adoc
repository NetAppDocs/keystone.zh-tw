= 修改雲端服務詳細資料
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
使用此方法修改雲端服務的詳細資料。您可以：

* 修改每項服務的詳細資料、如下表所列。
* 新增或移除服務的標記。


|===
| 雲端平台 | 您可以修改： 


| Azure | Azure 用戶端 ID 、 Azure 用戶端機密與名稱 


| Azure 美國政府 | TBA 


| GCP | TBA 


| AWS | TBA 
|===
.步驟
. 從功能表中選取 * Cloud Services* > * Cloud Services* 。
. 在清單中找出雲端服務、然後按一下「編輯」圖示。
. 在「更新雲端服務」對話方塊中、視需要進行任何變更。
. 按一下 * 更新 * 。
