= 檢視雲端服務清單
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Cloud Services 清單會顯示所選租戶中所有雲端服務的清單。

若要檢視清單、請從功能表中選取 * Cloud Services* > * Cloud Services* 。

清單會顯示每個雲端服務的簡單資訊。

每個雲端服務旁的行動圖示可讓您修改或刪除雲端服務。
